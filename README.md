## Installation
Github:

    git clone https://github.com/I2rys/DAD
    
NPM Packages:
```
npm i request
```

## Usage
```
node index.js <discord_token> <proxies>
```

+ discord_token - The Discord account token to disable.
+ proxies - Your file path that contains HTTP/HTTPS proxies.
